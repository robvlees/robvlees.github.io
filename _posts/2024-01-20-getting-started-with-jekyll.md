Jekyll is a powerful static site generator that's perfect for blogs, portfolios, and documentation sites. In this tutorial, I'll show you how to get started.

## Why Jekyll?

Jekyll offers several advantages:

- **Fast** - Static HTML means lightning-fast page loads
- **Secure** - No database or server-side code to hack
- **Free Hosting** - Deploy directly to GitHub Pages
- **Markdown Support** - Write content in Markdown

## Installation

First, make sure you have Ruby installed. Then install Jekyll:

```bash
gem install jekyll bundler
```

Create a new site:

```bash
jekyll new my-site
cd my-site
bundle exec jekyll serve
```

## Conclusion

Jekyll is an excellent choice for building fast, secure websites. Give it a try!
